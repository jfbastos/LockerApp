##### Senioridade: Pleno

##### Tecnologias utilizadas:
- Arquitetura MVI
- Ui em Compose
- Injeção de dependência utilizando Koin

##### Configurações do Projeto
- Target SDK 36
- Min. SDK 26
- Java 11

##### Features implementadas 

- SignIn e SignUp
- Listagem de portas e buscar por estas

##### Uso de IA no projeto

A IA utilizada foi a [Cloude.ia](https://claude.ai/new)

- Utilizada para facilitar na criação de alguns models dado retorno da API;
- Na tela de listagem, utilizado para encontrar uma lógica alternativa ao Paging3,
visto que o mesmo acaba se tornando muito verboso ou complexo para cenários mais simples;
- Utilizado para auxiliar na criação e validação de alguns testes unitários;
- Consulta geral de documentações e implementações (alguns componentes do compose e possibilidades de customização).
- Criação de alguns cenários de testes unitários.
